Notifies the administrator about posts that need updating based on a configurable time threshold.

== Description ==

This plugin notifies the site administrator about posts that have not been updated for a configurable period of time. It is ideal for keeping website content up-to-date and relevant, which can improve SEO and user experience.

== Installation ==

1. Upload the plugin file to the `/wp-content/plugins/` folder.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings > Update Reminder' and configure the time threshold.

== Usage ==

- Set the time threshold in months.
- Receive notifications in the admin panel about outdated posts.
- Click on the links to edit the posts that need updating.

== Frequently Asked Questions ==

= Can I change the time threshold? =
Yes, the time threshold is configurable on the plugin settings page.
